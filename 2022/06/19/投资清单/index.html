投资清单

2022-06-19
估值理论

投资清单

了解企业-企业的基本要素

  1. 你是否愿意花很多时间去了解这家企业？
  2. 如果你是首席执行官，你会如何评估这家企业？
  3. 你能用自己的语言描述企业的运作方式吗？
  4. 企业如何赚钱？
  5. 随着时间的推移，企业如何发展的？
  6. 企业在哪些国外市场中运营，有何风险？

了解企业-从客户的角度

  1. 企业的核心客户是谁？
  2. 客户群是集中化还是多元化的？
  3. 劝说客户购买其产品或服务，是容易还是困难？
  4. 企业的客户留存率是多少？
  5. 企业以客户为导向的标志是什么？
  6. 企业为客户解决了哪些问题？
  7. 客户对企业产品或服务的依赖程度如何？
  8. 如果某家企业明天消失，会对客户群造成什么影响？

评估企业和行业的优劣势

  1. 企业是否具有可持续的竞争优势，其来源是什么？
  2. 企业是否具备提高价格却不失去客户的能力？
  3. 企业所在的行业是好还是坏？
  4. 随着时间的推移，行业是如何演变的？
  5. 企业的竞争格局是什么样的，竞争有多激烈？
  6. 企业与供应商之间建立了怎样的关系？

衡量企业运营和财务状况

  1. 企业的基本面是什么？
  2. 你需要监控哪些企业的运营指标？
  3. 企业面临的主要风险是什么？
  4. 通货膨胀会对企业造成什么影响？
  5. 企业的资产负债表是强还是弱？
  6. 企业的投入资本回报率是多少？

评估收益分配（现金流）

  1. 管理层采用保守还是自由的会计标准？
  2. 企业产生经常性收益还是一次性交易收益？
  3. 企业周期性、逆周期性和抗衰退型的程度如何？
  4. 经营杠杆影响企业收益的程度如何？
  5. 营运资本如何影响企业的现金流？
  6. 企业资本支出需求是高还是低？

评估管理层素质-背景和类型：他们是谁

  1. 领导公司的管理者是什么类型？
  2. 引进外部管理层会对企业产生哪些影响？
  3. 管理者是狮子型管理者，还是鬣狗型管理者？
  4. 管理者如何上升到领导整个企业的位置？
  5. 高级管理层如何获得报酬和所有权？
  6. 管理者是否一直在买入或卖出股票？

评估管理层素质-能力：管理层如何运营企业

  1. 首席执行官管理企业时是否惠及所有股东？
  2. 管理层是日益改善运作，还是使用战略计划来引领企业？
  3. 首席执行官和首席财务官是否就盈利问题发布指导意见？
  4. 企业管理层是以集中还是分散的方式进行管理？
  5. 管理层是否重视员工？
  6. 管理层是否知道如何很好地招聘？
  7. 管理层是否关注削减不必要的成本？
  8. 首席执行官和首席财务官在制定资本分配决策时是否纪律严明？
  9. 首席执行官和首席财务官是否会适时回购股票？

评估管理层素质-积极和消极特质

  1. 首席执行官爱钱还是爱企业？
  2. 你能否识别管理人员是否正直？
  3. 管理者对股东的言行是否明确和一致？
  4. 管理层是否会独立思考，并不受到同行者行为的影响？
  5. 首席执行官是否会自我推销？

评估增长机会

  1. 企业是通过并购（M & A）增长，还是有机增长？
  2. 管理层让企业增长的动机是什么？
  3. 历史增长是否有利可图，是否会持续下去？
  4. 企业未来的增长前景如何？
  5. 管理层是让企业过快还是稳步增长？

评估合并和收购

  1. 管理层如何做出合并与收购的决策？
  2. 企业过去的收购是否成功？
